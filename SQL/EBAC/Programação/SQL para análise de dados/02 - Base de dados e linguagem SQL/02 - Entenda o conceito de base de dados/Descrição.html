Os materiais de apoio do curso são "notebooks", ou seja, arquivos com extensão IPYNB que devem ser abertos na plataforma do Google chamada Google Colab. Para isso, faça o download do material em sua máquina. Acesse a conta da Google. Vá em https://colab.research.google.com/. No item "Arquivo", escolha a opção "Abrir notebook" e na área "Upload" escolha o arquivo que fez o download.
