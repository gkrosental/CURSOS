Na quarta aula, você aprendeu sobre alterar e descartar tabelas. Agora, dê play nessa videoaula para aprender a teoria sobre inserir, atualizar e apagar dados, aprofundando os seguintes conceitos: 

  • compreender as definições básicas de inserção de dados em uma tabela;
  • aprender sobre atualização de dados e como ela afeta as tabelas;
  • compreender o processo de exclusão de dados em uma tabela e seus efeitos na estrutura de dados.
