Na quinta aula, você aprendeu sobre a teoria do uso da seleção condicional com case/when/then. Agora, dê play nessa videoaula para a parte prática, aprofundando os seguintes conceitos: 

  • utilizar a estrutura CASE/WHEN/THEN em consultas SQL para realizar seleções condicionais;
  •  criar consultas SQL utilizando a estrutura CASE/WHEN/THEN para selecionar dados com base em condições específicas;
  •  comparar a estrutura CASE/WHEN/THEN com outras opções de seleção condicional em SQL.
