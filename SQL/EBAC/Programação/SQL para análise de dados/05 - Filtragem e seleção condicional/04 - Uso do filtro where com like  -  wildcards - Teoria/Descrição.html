Na segunda aula, você aprendeu sobre uso do filtro where com and/or/between. Agora, dê play nessa videoaula para entender sobre uso do filtro where com like/wildcards, aprofundando os seguintes conceitos: 

  • compreender o uso do operador LIKE para buscar padrões em dados textuais;
  • identificar os diferentes caracteres coringa (wildcards) disponíveis em SQL;
  • compreender como utilizar os caracteres curinga em conjunto com o operador LIKE para buscar padrões específicos.
