Na terceira aula, você aprendeu sobre a teoria do uso do filtro where com like/wildcards. Agora, dê play nessa videoaula para praticar esse processo, aprofundando os seguintes conceitos: 

  • utilizar o operador LIKE em consultas SQL para buscar padrões em dados textuais;

  • criar consultas SQL utilizando o operador LIKE e os caracteres curinga para buscar padrões específicos em dados textuais.
