Na quinta aula, você aprendeu sobre a teoria do uso da cláusula having. Agora, dê play nessa videoaula para a parte prática, aprofundando os seguintes conceitos: 

  • aplicar a cláusula having em consultas SQL para filtrar informações agregadas em uma tabela;
  • utilizar a cláusula group by para agrupar resultados de uma consulta em SQL e aplicar a cláusula having em conjunto com ela;
  • realizar consultas em uma tabela que utilize a cláusula having em conjunto com a cláusula group by.
