Na terceira aula, você aprendeu sobre a teoria das funções min/max/sum/avg. Agora, dê play nessa videoaula para praticar esse processo, aprofundando os seguintes conceitos: 

  • aplicar as funções min, max, sum e avg em consultas SQL para obter informações agregadas em uma tabela;
  • utilizar a cláusula group by para agrupar resultados de uma consulta em SQL e aplicar as funções min, max, sum e avg em conjunto com ela;
  • realizar consultas em uma tabela que utilize as funções min, max, sum e avg em conjunto com a cláusula group by.
