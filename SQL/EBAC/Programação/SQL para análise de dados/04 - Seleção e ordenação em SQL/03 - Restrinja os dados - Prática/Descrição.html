Na primeira aula, você estudou sobre a teoria de restrição de dados. Agora, dê play nessa videoaula para praticar esse processo, aprofundando os seguintes conceitos: 

  • criar restrições de dados em um banco de dados utilizando SQL;
  • verificar a validação de restrições de dados ao inserir, atualizar ou excluir dados;
  • resolver problemas de restrições de dados que possam surgir durante a manipulação de dados.
