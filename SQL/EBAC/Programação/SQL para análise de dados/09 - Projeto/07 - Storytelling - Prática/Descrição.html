Na aula anterior, você aprendeu sobre Análise de dados. Agora, dê play nessa videoaula para praticar o processo de Storytelling, aprofundando os seguintes conceitos: 

  • compreender a importância da comunicação de dados por meio de storytelling;
  • identificar o público-alvo e adaptar a narrativa dos dados de acordo com o perfil do público;
  • utilizar ferramentas de visualização de dados para criar apresentações impactantes e envolventes.
