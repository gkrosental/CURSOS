1. Crie uma pasta no seu computador para guardar os seus arquivos dos projetos de estudo. Depois disso, abra essa pasta com o VS Code.
  2. Dentro dessa pasta crie um arquivo chamado index.htmlObs.: atente-se para a extensão .html, ela é importante para definir o formato do arquivo.Obs².: o nome index é utilizado para definir o arquivo principal de um diretório.
  3. No arquivo index.html adicione o seguinte código: 
<html>
<head>
	<title>Aprendendo HTML</title>
</head>

<body>
	<h1>Olá, mundo!</h1>
	Bora dominar a linguagem HTML!
</body>
</html>
  1. Agora abra esse arquivo com o seu navegador.
  2. Outra caminho para visualizar rapidamente os arquivos HTML no navegador é instalando a extensão Live Server do VS Code. Ela permite criar um servidor local que entrega os arquivos do diretório de trabalho na URL http://localhost:5500/ e ainda atualiza automaticamente a página sempre que salvarmos alguma alteração no código.
  3. Altere o arquivo index.html e veja que a página se atualiza automaticamente no navegador: 
<html>
<head>
	<title>Aprendendo HTML</title>
</head>

<body>
	<h1>Olá, mundo!</h1>
	<h2>Bora dominar a linguagem HTML!</h2>
</body>
</html>
