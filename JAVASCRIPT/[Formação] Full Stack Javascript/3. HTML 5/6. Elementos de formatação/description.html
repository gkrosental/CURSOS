Crie um novo arquivo para essa aula chamado “formatacao.html” com o seguinte conteúdo: 
<html>
<head>
	<title>Elementos de Formatação</title>
</head>

<body>
	<h1>Elementos de Formatação</h1>
	<p>
		No HTML podemos escrever utilizando negrito,
		itálico ou ambos para destacar partes de um texto.
	</p>
</body>
</html>
Agora utilize as tags <b> e <i> para negrito e itálico respectivamente: 
<html>
<head>
	<title>Elementos de Formatação</title>
</head>

<body>
	<h1>Elementos de Formatação</h1>
	<p>
		No HTML podemos escrever utilizando <b>negrito</b>,
		<i>itálico</i> ou <b><i>ambos</i></b> para destacar partes de um texto.
	</p>
</body>
</html>
No entanto, existem tags semânticas para essa mesma função, o <strong> e o <em>: 
<html>
<head>
	<title>Elementos de Formatação</title>
</head>

<body>
	<h1>Elementos de Formatação</h1>
	<p>
		No HTML podemos escrever utilizando <strong>negrito</strong>,
		<em>itálico</em> ou <strong><em>ambos</em></strong> para destacar
		partes de um texto.
	</p>
</body>
</html>
As tags <b> e <i> são meramente para estilização e devem ser evitadas porque essa estilização deve ser feita preferencialmente com CSS, que você irá aprender no próximo módulo. Já as tags <strong> e <em> devem ser utilizadas pelo seu significado, destaque e ênfase, independente dos estilos aplicados.
