No HTML podemos também utilizar o elemento <br> para fazer quebra de linha (pular linha) manualmente: 
<html>
<head>
    <title>Quebra de linha e régua horizontal</title>
</head>

<body>
    <h1>Quebra de linha e régua horizontal</h1>

    <p>Primeira linha.<br>Segunda linha.<br>Terceira linha.</p>

    <p>Quarta linha.<br>Quinta linha.<br>Sextalinha.</p>
</body>
</html>
Além disso, podemos usar um elemento chamado de régua horizontal, ou horizontal rule, através da tag <hr> para criar uma linha separadora na página: 
<html>
<head>
    <title>Quebra de linha e régua horizontal</title>
</head>

<body>
    <h1>Quebra de linha e régua horizontal</h1>

    <p>Primeira linha.<br>Segunda linha.<br>Terceira linha.</p>

		<hr>

    <p>Quarta linha.<br>Quinta linha.<br>Sextalinha.</p>
</body>
</html>
É importante destacar que, apesar de ser possível utilizar esses elementos, muitas vezes esses efeitos (quebra de linha e separação através de uma linha horizontal) serão obtidos através do CSS, o que faz com que essas tags sejam pouco usadas (especialmente o <br>). Apesar disso, você pode sim utilizá-las se necessário e não há nada de errado com isso.
