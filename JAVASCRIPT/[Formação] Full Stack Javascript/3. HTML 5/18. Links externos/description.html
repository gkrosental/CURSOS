No mesmo arquivo da aula de links, crie um link externo que leva para o Google. Repare que para isso precisamos especificar o protocolo “http://” no começo do caminho, se não o fizéssemos o navegador entenderia que se trata de uma URL relativa: 
<html>
<head>
	<title>Trabalhando com links</title>
</head>

<body>
	<h1>Trabalhando com links</h1>

	<p>
		Navegue entre os arquivos das aulas:
		<a href="/">Primeira página</a>
		<a href="titulos.html">Aula: títulos e parágrafos</a>
		<a href="formatacao.html">Aula: elementos de formatação</a>
		<a href="imagens.html">Aula: imagens</a>
	</p>

	<p>Acessar a página do <a href="https://google.com">Google</a>.</p>
</body>
</html>
Uma técnica interessante para se aplicar nas situações de links para outros sites e abri-los em uma nova guia, garantindo que o usuário conseguirá voltar facilmente para o seu site: 
<html>
<head>
	<title>Trabalhando com links</title>
</head>

<body>
	<h1>Trabalhando com links</h1>

	<p>
		Navegue entre os arquivos das aulas:
		<a href="/">Primeira página</a>
		<a href="titulos.html">Aula: títulos e parágrafos</a>
		<a href="formatacao.html">Aula: elementos de formatação</a>
		<a href="imagens.html">Aula: imagens</a>
	</p>

	<p>Acessar a página do <a href="https://google.com" target="_blank">Google</a>.</p>
</body>
</html>
