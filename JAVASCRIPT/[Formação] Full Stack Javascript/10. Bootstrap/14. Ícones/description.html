1. Os ícones são muito extensos, então não conseguimos colocar aqui na documentação da OBC, mas você pode ver eles através desse link: https://icons.getbootstrap.com/ e com isso você consegue instalar com o comando que tem logo na primeira parte da página, que nem fizemos com o Sass
2. Colocando esse comando no terminal você terá os ícones do Bootstrap disponíveis para uso. 
​npm i bootstrap-icons
3. Você pode pesquisar pelo ícone que você quer aqui na parte de pesquisa, podemos usar o exemplo do Instagram, que você vai clicar nele, sendo redirecionado para outra página, e vai ver que na lateral você vai ter o código desse ícone. 
<i class="bi bi-instagram"></i>
4. Logo abaixo você vê também exemplos de itens do Bootstrap usando dentro deles os ícones, como o botão, o heading, etc... 
<button type="button" class="btn btn-primary"> <i class="bi bi-instagram"></i> Primary</button>
5. Você tem também a opção de pegar o SVG da imagem, para manipular como você quiser como uma imagem qualquer.
