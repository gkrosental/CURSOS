1. Você vai primeiro acessar o site: https://michalsnik.github.io/aos/ , com isso você vai ver os elementos, vamos usar eles em textos.
2. Faremos a instalação, basta clicar no ícone do github e copiar o link fornecido lá 
<link rel="stylesheet" href="https://unpkg.com/aos@next/dist/aos.css" />
3. Vamos primeiro fazer um style para que os textos eles tenham diferentes momentos de exibição. 
.div1{
 height: 100vh;
}
.div2{
 height: 100vh;
}
4. Com isso podemos criar textos aqui nas divs, colocando eles dentro dessas divs, fazendo com que eles ganhem as características de animação. 
<div data-aos="fade-up"></div>

<div data-aos="fade-left"></div>

<div data-aos="flip-left"></div>

<div data-aos="zoom-in"></div>
