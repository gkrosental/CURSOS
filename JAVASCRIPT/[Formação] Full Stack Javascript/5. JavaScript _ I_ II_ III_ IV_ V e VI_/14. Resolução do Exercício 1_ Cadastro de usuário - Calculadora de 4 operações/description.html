Agora vamos criar o script e começar obtendo as informações do recruta usando o prompt: 
const primeiroNome = prompt("Informe o primeiro nome do recruta:")
const sobrenome = prompt("Informe o sobrenome do recruta:")
const campoDeEstudo = prompt("Qual é o campo de estudo do recruta?")
const anoDeNascimento = prompt("Qual é o ano de nascimento do recruta?")
E então exibimos essas informações usando o alert e ajustando o texto do nome completo e calculando a idade baseada no ano: 
alert(
  "Recruta cadastrado com sucesso!\n" +
  "\nNome completo: " + primeiroNome + " " + sobrenome +
  "\nCampo de estudo: " + campoDeEstudo +
  "\nIdade: " + (2022 - anoDeNascimento)
)

Calculadora de 4 Operações

  1. Começamos criando o arquivo HTML e incluindo o script:
Depois de criar o script, vamos começar obtendo os dois números usando o prompt: 
const entrada1 = prompt("Informe o primeiro número:")
const entrada2 = prompt("Informe o segundo número:")

const x = parseFloat(entrada1)
const y = parseFloat(entrada2)
Com os números vamos calcular os resultados das 4 operações: 
const soma = x + y
const subtracao = x - y
const multiplicacao = x * y
const divisao = x / y
Por fim, vamos exibir os resultados na tela de forma organizada: 
alert(
  "Resultados:\n" +
  "\nSoma: " + soma +
  "\nSubtração: " + subtracao +
  "\nMultiplicação: " + multiplicacao +
  "\nDivisão: " + divisao
)
