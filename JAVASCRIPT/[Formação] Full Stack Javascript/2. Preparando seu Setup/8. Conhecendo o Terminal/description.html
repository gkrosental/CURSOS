Conhecendo o Terminal

  1. O terminal é a forma mais simples de interagir com o SO
  2. Com ele podemos enviar comandos diretamente para o SO
  3. Muitas das tarefas que fazemos através da interface gráfica podem ser feitas pelo terminal, como acessar pastas, abrir arquivos, executar programas, etc
  4. Comandos essenciais pwd: exibe o caminho do diretório atualls: lista o conteúdo do diretório atualmkdir: cria um diretório vaziocd: muda de diretórioni ou touch: cria um arquivo vaziocp: copia um arquivomv: move um arquivorm: remove um arquivo ou diretório
  5. pwd: exibe o caminho do diretório atual
  6. ls: lista o conteúdo do diretório atual
  7. mkdir: cria um diretório vazio
  8. cd: muda de diretório
  9. ni ou touch: cria um arquivo vazio
  10. cp: copia um arquivo
  11. mv: move um arquivo
  12. rm: remove um arquivo ou diretório
