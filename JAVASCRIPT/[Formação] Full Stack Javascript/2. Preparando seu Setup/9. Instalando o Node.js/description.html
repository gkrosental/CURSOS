Instalando o Node.js e o npm

Instalação no Windows e MacOS

  1. Baixar o instalador de acordo com a versão do seu sistema operacional em https://nodejs.org/pt-br/download/
  2. Executar o instalador e seguir as instruções (a instalação padrão deve ser suficiente)

Instalação no Linux

Debian/Ubuntu e derivadas (Pop!_OS, Linux Mint, Zorin, Elementary, etc) 

sudo apt install -y nodejs

Arch Linux e derivadas (Manjaro, EndeavourOS 

pacman -S nodejs npm

Fedora 

sudo dnf install -y curl

Instalação com o gerenciador de versões asdf (Linux e MacOS)

  1. Siga os passos na documentação oficial disponível em https://asdf-vm.com/pt-br/guide/getting-started.html
