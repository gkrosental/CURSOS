Conhecendo o Vite

  1. Um servidor de desenvolvimento que trabalha com ESM para fornecer Hot Module Replacement (HMR)
  2. Um empacotador (Rollup) que já vem pré-configurado para gerar assets otimizados para produção
  3. Mais rápido
  4. Mais simples
  5. Criar um novo projeto com Vite é extremamente simples, basta executar:
  6. npm create vite@latest
