Instalação no Windows
  1. Vá até o Centro de Downloads do MongoDB: https://www.mongodb.com/try/download/community
  2. Role pela página até encontrar o botão "Download" e clique para baixar o instalador.
  3. Encontre o arquivo baixado no seu computador e execute-o para iniciar a instalação.
  4. Siga os passos da instalação (a instalação padrão deve ser suficiente). Ao finalizar, você deverá ter o MongoDB instalado juntamente com o MongoDBCompass, um software de interface gráfica para acessar seus bancos de dados.
  5. É possível utilizar o mongosh (linha de comando do mongo) dentro do MongoDBCompass, basta se conectar ao servidor local (conexão padrão, em mongodb://localhost:27017) e abrir o mongosh na barra inferior da interface.
  6. Opcional: caso queira utilizar o mongosh pelo terminal do Windows você pode fazer o download separadamente pelo Centro de Downloads do MongoDB: https://www.mongodb.com/try/download/shell
Instalação no Linux
  1. Acesse: https://www.mongodb.com/docs/manual/administration/install-on-linux/
  2. Selecione a sua distribuição Linux e siga os passos para a instalação do pacote adequado.
Instalação no Mac
  1. Siga os passos da documentação do MongoDB em: https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-os-x/
