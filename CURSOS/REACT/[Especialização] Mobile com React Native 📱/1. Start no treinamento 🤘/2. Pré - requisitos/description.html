Pré-requisitos

Esse é um curso avançado e alguns conhecimentos prévios são necessários. Antes de começar, você precisa ter conhecimentos básicos de programação, como: estrutura de dados, lógica de programação, conhecimentos na linguagem Javascript e em React. Nesse curso de React Native, utilizamos muitos desses conceitos e com certeza se você não tiver algum conhecimento prévio, terá mais dificuldades e em algum momento precisará voltar para entender esses conceitos básicos. Você não precisa conhecer profundamente esses conceitos, mas você precisa ter uma noção de como as coisas funcionam.
Caso você não tenha conhecimento de lógica, Javascript e React, temos um treinamento especial para você: o FullStack Javascript Profissional, onde você vai aprender todos esses conceitos do Zero ao profissional. Com certeza será um complemento muito interessante caso você esteja iniciando. Além disso, se você quiser melhorar seus conhecimentos em lógica, temos também um treinamento exclusivo para desenvolver suas habilidades em lógica: 30 desafios Javascript, onde você vai poder praticar bastante o Javascript e a lógica com desafios desde o nível very easy até o nível very hard.
