E aii programador! Seja bem-vindo ao curso de React Native da OneBitCode!
Nesse curso, você vai entrar completamente no mundo do mobile. Vamos falar sobre carreira, mercado, e além disso, você vai aprender os principais fundamentos do React Native, com aulas teóricas, práticas, quizzes e exercícios, entendendo como ele funciona e aplicando os conceitos em projetos.
Teremos 2 projetos dentro do curso: o primeiro, será para aplicar todos os conceitos teóricos aprendidos, um exercício completo, onde vamos desenvolver um game de uma bomba relógio! O segundo, será o nosso projeto principal, que é um marketplace. Ele terá todas as funcionalidades aprendidas no curso e mais alguns conceitos específicos desse tipo de projeto.
Vamos utilizar o Expo para o desenvolvimento dos aplicativos e você também vai conhecer os fundamentos e aplicações dele, na prática.
Além disso, depois de desenvolver o Marketplace no projeto final, vai aprender como disponibilizar nas lojas (Play Store e Apple Store) para download.
É o primeiro passo para você começar a desenvolver projetos para empresas e disponibilizar seus próprios projetos nas lojas de app! Com todo o conteúdo do curso, você estará pronto para entrar no mercado mobile com React Native e Expo!
Participe da nossa comunidade no Discord, e caso você tenha alguma dúvida durante o curso, os instrutores e colegas de turma estarão lá para te ajudar. Siga as regras da comunidade e tire todas as suas dúvidas para melhor aproveitamento do curso.
